# README #

### What is this repository for? ###

Erg Explorer (Alpha) project repo

### How do I get set up? ###

Install Node.JS 11.15.0

npm install -g bower
npm install --global gulp-cli
npm install
bower install
npm install natives@1.1.6

### Commands ###

gulp serve
Starts local server host on localhost:9000

gulp build
Builds the website
